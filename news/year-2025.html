2025-12-06 - StainX: CUDA-optimized stain normalization is now open source!

We are excited to release StainX, a Python package for enhanced stain normalization of histopathology images with batch processing support.

StainX provides multiple algorithms (Reinhard, Macenko, Histogram Matching) with automatic backend selection for CPU, GPU (CUDA), and Apple Silicon (MPS) devices.

Our preliminary benchmarks show 8-11x speedups compared to standard PyTorch implementations when using optimized CUDA kernels!

Install it with pip install stainx and check out the GitHub repository.

2025-03-12 - LazySlide and WSIData are now part of the scverse ecosystem!

LazySlide and WSIData are now part of the scverse ecosystem, and are being integrated into the scientific workflow.

WSIData is a Python package for working with whole slide images (WSI) using scverse-compatible interfaces, and LazySlide provides the tools and operations needed to analyze WSIs.

The official acceptance into the scverse ecosystem will enable scientists from all backgrounds but in particular from the single-cell and genomics fields which are already using scverse tools to seemlessly work with WSIs as if it were any other single-cell type of data.

We are looking forward to upcoming stable releases and manuscript which will present the scverse software stack for WSI analysis.

2025-02-06 - Our lab's project template is now open source!

We're excited to share that we've open-sourced our project template, designed to make starting new projects a breeze and homogeneous across all our projects.

This template uses Cookiecutter, a widely-used tool that helps create projects from templates.

Our template is flexible, letting you choose options like project type (research or software), package management (uv, pip, conda), and your preferred code editors we most use (Sublime Text or VSCode).

By sharing we hope to help others in the scientific community organize their projects effectively and also motivate others to share their ways of structuring research and software projects.
